# 通讯录管理系统

## 项目描述

一个基于C++的个人通讯录管理程序，支持联系人信息的添加、删除、修改、查询和分类管理。

## 功能特性

### 核心功能

1. **添加联系人** - 添加新的联系人信息（姓名、电话、邮箱、地址、分类）
2. **删除联系人** - 根据电话号码查找并删除联系人
3. **修改联系人** - 修改现有联系人的信息
4. **查询联系人** - 支持多种查询方式：
   - 按姓名查询
   - 按电话查询
   - 按邮箱查询
   - 按地址查询
   - 按分类查询
   - 综合查询（所有字段）

### 管理功能

5. **显示所有联系人** - 显示通讯录中所有联系人
6. **按分类显示** - 显示指定分类下的所有联系人
7. **显示所有分类** - 列出所有分类及每个分类的联系人数量

### 数据持久化

8. **保存数据到文件** - 将通讯录数据保存到 `address_book.txt`
9. **从文件加载数据** - 从文件加载之前保存的数据

## 联系人信息字段

- **姓名** (name)
- **电话** (phone) - 作为唯一标识符
- **邮箱** (email)
- **地址** (address)
- **分类** (category) - 如：家人、朋友、同事、其他等

## 编译和运行

### 编译

```bash
g++ -o address_book address_book.cpp
```

### 运行

```bash
./address_book
```

Windows系统：

```bash
g++ -o address_book.exe address_book.cpp
address_book.exe
```

## 使用说明

1. 程序启动时会自动尝试加载 `address_book.txt` 文件（如果存在）
2. 根据菜单提示选择相应功能
3. 退出程序时会自动保存数据到文件

## 注意事项

- 电话号码作为唯一标识符，不能重复
- 数据以文本格式保存在 `address_book.txt` 文件中
- 支持中文输入和显示
